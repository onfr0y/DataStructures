# C++ Data Structures Practice

This directory contains C++ implementations and practices for various data structures. It is configured as a CMake project for ease of compilation and testing.

## Directory Structure

* `CMakeLists.txt` - CMake build configuration
* `main.cpp` - Singly Linked List template implementation and demonstration

## Build Instructions

Using Clang or GCC:

```bash
# Compile the main application
clang++ -std=c++20 main.cpp -o main

# Run the practice executable
./main
```
